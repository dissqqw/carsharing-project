Инструкция по запуску проекта
1. Установка программ
Скачай и установи:

Python — https://www.python.org/downloads/

Нажми жёлтую кнопку «Download Python»

При установке обязательно поставь галочку «Add Python to PATH»

Node.js — https://nodejs.org/

Нажми зелёную кнопку «LTS»

Устанавливай с настройками по умолчанию

Перезагрузи компьютер после установки.

2. Проверка установки
Открой Command Prompt (нажми Win+R, введи cmd, нажми Enter).

Проверь Python:

text
python --version
Должен показать версию, например Python 3.12.0.

Проверь Node.js:

text
node --version
Должен показать версию, например v20.11.0.

3. Скачивание проекта
Скачай архив с проектом и распакуй его. Например, в папку C:\drivehub.

4. Открытие папки проекта
В Command Prompt перейди в папку проекта:

text
cd C:\drivehub
(замени C:\drivehub на свой путь, куда распаковал проект)

5. Установка Python-зависимостей (бэкенд)
В этом же окне Command Prompt (ты должен быть в папке C:\drivehub):

text
pip install -r requirements.txt
Дождись окончания установки. Появится Successfully installed ....

6. Запуск сервера
В этом же окне:

text
python app.py
Появится сообщение:

text
Сервер запущен на http://127.0.0.1:5000
Это окно не закрывай. Оставь его работать.

7. Установка фронтенда
Открой второе окно Command Prompt (Win+R, cmd, Enter).

Перейди в папку frontend:

text
cd C:\drivehub\frontend
Установи зависимости:

text
npm install
Дождись окончания.

8. Запуск фронтенда
В этом же втором окне:

text
npm run dev
Появится ссылка:

text
http://localhost:5173
Открой браузер и перейди по этой ссылке.

9. Вход
Администратор:

Имя, Фамилия — любые

Дата рождения — любая

Роль — Администратор

Ключ доступа: drivehub-admin-2026

Пользователь:

Имя, Фамилия — любые

Дата рождения — любая

Роль — Пользователь

Важно
Первое окно (сервер) — не закрывай, пока пользуешься приложением

Второе окно (фронтенд) — не закрывай, пока пользуешься приложением

Если закроешь окно — сервер или фронтенд остановятся, приложение перестанет работать
