Как запустить проект DriveHub у себя на компьютере
Что нужно установить заранее
Python — скачай с https://www.python.org/downloads/ (жёлтая кнопка). При установке ОБЯЗАТЕЛЬНО поставь галочку «Add Python to PATH».

Node.js — скачай с https://nodejs.org/ (зелёная кнопка LTS). Устанавливай с настройками по умолчанию.

После установки перезагрузи компьютер.

Шаг 1: Скачай проект
Тебе скинут ZIP-архив carsharing_project-main.zip. Распакуй его. Получится папка carsharing_project-main. Запомни, куда ты её распаковала (например, C:\Users\ТвоёИмя\Downloads\carsharing_project-main).

Шаг 2: Открой папку в VS Code
Открой VS Code

Нажми File → Open Folder

Найди папку carsharing_project-main и выбери её

Нажми Выбор папки

Шаг 3: Открой терминал
В VS Code нажми Terminal → New Terminal (или Ctrl + ~). Внизу появится окно терминала.

Важно: убедись, что в терминале написано carsharing_project-main. Если нет — напиши команду:

text
cd путь_к_папке
Например:

text
cd C:\Users\ТвоёИмя\Downloads\carsharing_project-main
Шаг 4: Установи Python-зависимости
В терминале напиши и нажми Enter:

text
pip install -r requirements.txt
Дождись, пока всё установится. Появится Successfully installed ....

Шаг 5: Запусти сервер
В этом же терминале напиши:

text
python app.py
Появится надпись:

text
Сервер запущен на http://127.0.0.1:5000
НЕ закрывай это окно терминала! Оставь его работать.

Шаг 6: Открой ВТОРОЙ терминал
В VS Code нажми на плюсик в правом верхнем углу панели терминала. Откроется второе окно терминала.

В нём перейди в папку frontend:

text
cd frontend
Шаг 7: Установи зависимости фронтенда
Во втором терминале напиши:

text
npm install
Дождись окончания.

Шаг 8: Запусти фронтенд
В том же втором терминале напиши:

text
npm run dev
Появится ссылка:

text
http://localhost:5173
Открой браузер (Chrome, Firefox) и перейди по этой ссылке.

Шаг 9: Вход в приложение
На странице входа:

Администратор:

Введи любые имя и фамилию

Выбери любую дату рождения

Переключатель «Администратор»

Ключ доступа: drivehub-admin-2026

Нажми «Войти»

Пользователь:

Введи любые имя и фамилию

Выбери любую дату рождения

Переключатель «Пользователь»

Нажми «Войти»

Важно
Первый терминал (сервер) должен работать всё время

Второй терминал (фронтенд) должен работать всё время

Если закроешь терминал — соответствующая часть перестанет работать

Чтобы остановить — нажми Ctrl+C в терминале
